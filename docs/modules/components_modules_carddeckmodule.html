Module components/modules/CardDeckModule

This file contains the TinaCMS configuration and the HTML rendering of the Card Deck block. The user can customize the image, title, subtitle, content, and link of each card. Additionally, the user can choose to have the image on the top or bottom, have the link as a button or link, and can also choose to get rid of the link altogether. The user can add/delete cards as they see fit.
