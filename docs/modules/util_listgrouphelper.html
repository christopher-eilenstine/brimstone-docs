syncParentModule

  • syncParentModule(page: SiteSubpage, parent: SitePage, form: DynamicObject): string
    • Defined in src/util/ListGroupHelper.tsx:77

    !! Warning! This function mutates the parent object! !!

    Given a subpage, this function finds its subpage container entry in its given parent page modules and (if necessary) updates it using the values on the given form.

    Parameters

    • page: SiteSubpage

      A subpage object, treated as non-mutable.

    • parent: SitePage

      A mutable page object.

    • form: DynamicObject

      A Tina CMS form, treated as non-mutable.

    Returns string

    The given page path when the subpage container entry is unchanged, otherwise the new page path for the given page.

syncSubpages

  • syncSubpages(page: SitePage, path: string, modules: DynamicObject[]): SiteSubpage[]
    • Defined in src/util/ListGroupHelper.tsx:27

    This function checks for the presence of subpage containers on the given page and (if present) updates or creates a subpage for each container entry. This function also removes subpages that no longer exist among the subpage container entries.

    Parameters

    • page: SitePage

      A page object.

    • path: string

      The relative path of the page being synced.

    • modules: DynamicObject[]

      The modules on the page being synced.

    Returns SiteSubpage[]

    An array of subpages analogous to each entry in all subpage containers on the given page (if any).
